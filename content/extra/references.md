## Haskell

- [Haskell Language Report](https://www.haskell.org/onlinereport/haskell2010/) — the official language standard.
- [Learn You a Haskell](http://learnyouahaskell.com/) — a beginner-friendly free online book.
- [Real World Haskell](http://book.realworldhaskell.org/) — a more practical introduction.
- [Haskell Wiki](https://wiki.haskell.org/) — community-maintained reference material.

## Cabal

- [Cabal documentation](https://cabal.readthedocs.io/) — official reference.
- [Hackage](https://hackage.haskell.org/) — the central Haskell package repository.

## GHCi

- [GHC User's Guide — GHCi](https://downloads.haskell.org/ghc/latest/docs/html/users_guide/ghci.html) — full reference for `.ghci` options.

## Functional Reactive Programming

- [Push-Pull Functional Reactive Programming](http://conal.net/papers/push-pull-frp/) — Conal Elliott's 2009 paper formalising FRP.
- [Functional Reactive Animation](https://conal.net/papers/icfp97/) — the original 1997 FRAN paper by Elliott & Hudak.

## Yampa

- [Yampa on Hackage](https://hackage.haskell.org/package/Yampa) — API documentation.
- [Yampa GitHub repository](https://github.com/ivanperez-keera/Yampa) — source code and issue tracker.
- [The Yampa Arcade](https://github.com/ivanperez-keera/Yampa/tree/develop/yampa-examples) — official example collection.
- [Functional Reactive Programming, Refactored](https://dl.acm.org/doi/10.1145/2976002.2976010) — Perez et al. 2016, the Dunai/BearRiver generalization.

## FRP in practice

- [Keera Hails](https://github.com/keera-studios/keera-hails) — a fully Yampa-based game/application framework.
- [sdl2 bindings](https://hackage.haskell.org/package/sdl2) — SDL2 for Haskell, the most practical choice for rendering alongside Yampa.
- [gloss](https://hackage.haskell.org/package/gloss) — simple 2D graphics library, easy to pair with Yampa for prototyping.
